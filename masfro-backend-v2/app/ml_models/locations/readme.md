1. Administrative Boundaries

    Type: Barangay

    Identifying Column: admin_level (value is "10")

    Description: The official boundary relations for the Barangays in Marikina.

2. Road Network

    Type: Roads

    Identifying Column: highway (e.g., residential, primary, service)

    Description: Any named road, street, or highway.

3. Landmarks & Services

    Type: Commercial (Malls/Markets)

        Identifying Columns: shop (mall, supermarket, market)

    Type: Public Services (Hospitals, Schools, Gov)

        Identifying Columns: amenity (hospital, school, university, townhall)

    Type: Recreational (Parks, Museums)

        Identifying Columns: leisure (park), tourism (museum)

4. Bridges (Critical Infrastructure)

    Type: Bridge

    Identifying Columns: bridge (yes), man_made (bridge)

    Description: Bridges over waterways (often used as flood gauges).

5. Residential Areas

    Type: Villages & Subdivisions

    Identifying Columns: landuse (residential), place (neighbourhood, quarter)

    Description: Named subdivisions, villages, or neighborhood areas.

6. Public Transport

    Type: Transport Hubs

    Identifying Columns: railway (station), public_transport (station), amenity (bus_station)

    Description: LRT stations, bus terminals, and jeepney stops.

7. Evacuation Centers

    Type: Sports Centers / Covered Courts

    Identifying Columns: leisure (sports_centre, pitch)

    Description: Gyms and covered basketball courts often used for evacuation. (Note: Schools are also evacuation centers but are captured under 'Public Services').

8. Waterways

    Type: Rivers & Creeks

    Identifying Column: waterway

    Description: Major rivers, streams, canals, and drains.
